    각 단원 목표¶

    1-1 (VM vs 컨테이너) VirtualBox와 Docker를 직접 설치하고, 같은 Nginx 서버를 두 방식으로 띄워 기동 시간·자원 사용량을 비교합니다.

    1-2 (기본 명령어) docker run, exec, ps, stop, rm, logs를 반복 사용하며 컨테이너 생명주기를 몸으로 익힙니다.

    1-3 (이미지 관리) 직접 빌드한 이미지를 Docker Hub에 push하고, 팀원 이미지를 pull해 실행할 수 있습니다.

    1-4 (볼륨) 컨테이너를 삭제해도 데이터가 유지되는 볼륨 구조를 이해하고, DB 컨테이너에 볼륨을 붙이는 실습을 합니다.

    1-5 (Dockerfile 최적화) 레이어 캐시 원리를 이해하고, 멀티스테이지 빌드로 이미지 크기를 50% 이상 줄입니다.

    1-6 (네트워크) 컨테이너 간 통신이 어떤 네트워크 위에서 이뤄지는지 이해하고, 커스텀 bridge 네트워크를 구성합니다.

    1-7 (Docker Compose) docker-compose.yml로 web·api·db 3개 서비스를 한 번에 정의·실행·종료하는 실무 패턴을 익힙니다.
