## 각 단원 목표

**1-1 (VM vs 컨테이너)**
VirtualBox와 Docker를 직접 설치하고, 같은 Nginx 서버를 두 방식으로 띄워 기동 시간·자원 사용량을 비교합니다.

**1-2 (기본 명령어)**
`docker run`, `exec`, `ps`, `stop`, `rm`, `logs`를 반복 사용하며 컨테이너 생명주기를 몸으로 익힙니다.

**1-3 (레지스트리와 이미지)**
공개 레지스트리(Docker Hub)와 프라이빗 레지스트리(ACR)의 차이를 이해하고, ACR에서 이미지를 pull해 컨테이너를 실행합니다. 레이어 캐시 구조도 직접 확인합니다.

**1-3-1 (나만의 이미지 만들고 ACR 올리기)**
Dockerfile을 직접 작성해 이미지를 빌드하고, 개인 ACR을 생성해 push/pull 전체 흐름을 실습합니다.

**1-4 (볼륨)**
컨테이너를 삭제해도 데이터가 유지되는 볼륨 구조를 이해하고, Docker Volume과 Bind Mount를 상황에 맞게 선택하는 실습을 합니다.

**1-5 (Dockerfile 최적화)**
레이어 캐시 원리를 이해하고, 멀티스테이지 빌드로 이미지 크기를 50% 이상 줄입니다.

**1-6 (네트워크)**
컨테이너 간 통신이 어떤 네트워크 위에서 이뤄지는지 이해하고, 커스텀 bridge 네트워크를 구성합니다.

**1-7 (Docker Compose)**
`docker-compose.yml`로 web·api·db 3개 서비스를 한 번에 정의·실행·종료하는 실무 패턴을 익힙니다.
