    1-6-2. лҸ„м „ кіјм ң : лҜёлӢҲ лё”лЎңк·ё¶

    кіјм ң к°ңмҡ”¶

    м§ҖкёҲк№Ңм§Җ л°°мҡҙ Docker л„ӨнҠёмӣҢнҒ¬ к°ңл…җмқ„ мҙқлҸҷмӣҗн•ҙм„ң, м»Ён…Ңмқҙл„Ҳл§ҢмңјлЎң лҸҷмһ‘н•ҳлҠ” лҜёлӢҲ лё”лЎңк·ё м„ңл№„мҠӨлҘј м§Ғм ‘ кө¬м„ұн•©лӢҲлӢӨ.

    лёҢлқјмҡ°м Җм—җм„ң кёҖмқ„ мһ‘м„ұн•ҳл©ҙ frontendк°Җ backendлЎң мҡ”мІӯмқ„ ліҙлӮҙкі , backendк°Җ лҚ°мқҙн„°лҘј JSON нҢҢмқјлЎң м ҖмһҘн•ң л’Ө лӢӨмӢң мқ‘лӢөн•ҳлҠ” кө¬мЎ°мһ…лӢҲлӢӨ. лӘЁл“  м»Ён…Ңмқҙл„ҲлҠ” м§Ғм ‘ docker run лӘ…л №мңјлЎң мӢӨн–үн•©лӢҲлӢӨ.

    м„ңл№„мҠӨ кө¬мЎ°¶

    мӮ¬мҡ©мһҗ лёҢлқјмҡ°м Җ
     вҶ“ (localhost:8080)
frontend (Nginx)
     вҶ“ (HTTP мҡ”мІӯ /posts)
backend (Flask)
     вҶ“
data.json м ҖмһҘ
    • frontend: Nginx кё°л°ҳ м •м Ғ нҺҳмқҙм§Җ м„ңл№ҷ + backendлЎң API мҡ”мІӯ н”„лЎқмӢң
    • backend: Flask кё°л°ҳ REST API, кІҢмӢңкёҖмқ„ data.jsonм—җ м ҖмһҘ/мЎ°нҡҢ
    • л‘җ м»Ён…Ңмқҙл„ҲлҠ” мӮ¬мҡ©мһҗ м •мқҳ л„ӨнҠёмӣҢнҒ¬лҘј нҶөн•ҙ нҶөмӢ 

    мӮ¬мҡ© мқҙлҜём§Җ¶

    м„ңл№„мҠӨ мқҙлҜём§Җ
    frontend skilleat/frontend:v4-kb5
    backend skilleat/backend:v4-kb5

    кіјм ң лӢЁкі„¶

    1лӢЁкі„: м „мҡ© л„ӨнҠёмӣҢнҒ¬ мғқм„ұ¶

    frontendмҷҖ backendк°Җ м„ңлЎң мқҙлҰ„мңјлЎң нҶөмӢ н•  мҲҳ мһҲлҸ„лЎқ мӮ¬мҡ©мһҗ м •мқҳ л„ӨнҠёмӣҢнҒ¬лҘј лЁјм Җ мғқм„ұн•©лӢҲлӢӨ.

    л„ӨнҠёмӣҢнҒ¬ мқҙлҰ„мқҖ мһҗмң лЎӯкІҢ м§Җм •н•ҙлҸ„ лҗ©лӢҲлӢӨ.

    2лӢЁкі„: backend м»Ён…Ңмқҙл„Ҳ мӢӨн–ү¶

    лӢӨмқҢ мЎ°кұҙмқ„ лӘЁл‘җ л§ҢмЎұн•ҳлҸ„лЎқ backend м»Ён…Ңмқҙл„ҲлҘј мӢӨн–үн•©лӢҲлӢӨ.

    • мқҙлҜём§Җ: skilleat/backend:v4-kb5
    • м»Ён…Ңмқҙл„Ҳ мқҙлҰ„: л°ҳл“ңмӢң backend-service
    • 1лӢЁкі„м—җм„ң мғқм„ұн•ң л„ӨнҠёмӣҢнҒ¬м—җ м—°кІ°
    • лӮҙл¶Җ нҸ¬нҠё: 5000лІҲ (нҳёмҠӨнҠём—җлҠ” л…ём¶ңн•ҳм§Җ м•Ҡм•„лҸ„ лҗ©лӢҲлӢӨ)

    мҷң backendлҘј лЁјм Җ мӢӨн–үн•ҳлӮҳмҡ”?

    frontendлҠ” мӢңмһ‘н•  л•Ң backend-serviceлқјлҠ” мқҙлҰ„мңјлЎң м—°кІ°мқ„ мӢңлҸ„н•©лӢҲлӢӨ. backendк°Җ лЁјм Җ мӢӨн–үлҗҳм–ҙ мһҲм–ҙм•ј frontendк°Җ м •мғҒм ҒмңјлЎң мһ‘лҸҷн•©лӢҲлӢӨ.

    3лӢЁкі„: frontend м»Ён…Ңмқҙл„Ҳ мӢӨн–ү¶

    лӢӨмқҢ мЎ°кұҙмқ„ лӘЁл‘җ л§ҢмЎұн•ҳлҸ„лЎқ frontend м»Ён…Ңмқҙл„ҲлҘј мӢӨн–үн•©лӢҲлӢӨ.

    • мқҙлҜём§Җ: skilleat/frontend:v4-kb5
    • backendмҷҖ лҸҷмқјн•ң л„ӨнҠёмӣҢнҒ¬м—җ м—°кІ°
    • нҳёмҠӨнҠё нҸ¬нҠё 8080 вҶ’ м»Ён…Ңмқҙл„Ҳ нҸ¬нҠё 80 мңјлЎң л°”мқёл”©

    4лӢЁкі„: мӣ№ м ‘мҶҚ л°Ҹ лҸҷмһ‘ нҷ•мқё¶

    лёҢлқјмҡ°м Җ лҳҗлҠ” curlлЎң м•„лһҳ мЈјмҶҢм—җ м ‘мҶҚн•©лӢҲлӢӨ.

    http://localhost:8080
    • лё”лЎңк·ё нҺҳмқҙм§Җк°Җ м •мғҒ н‘ңмӢңлҗҳлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ.
    • кёҖ мһ‘м„ұ нҸјм—җ м ңлӘ©кіј лӮҙмҡ©мқ„ мһ…л Ҙн•ҳкі  л“ұлЎқн•©лӢҲлӢӨ.
    • л“ұлЎқн•ң кёҖмқҙ нҷ”л©ҙм—җ м¶ңл ҘлҗҳлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ.

    5лӢЁкі„: м¶”к°Җ кІҖмҰқ¶

    backend лӮҙл¶Җ JSON нҢҢмқј нҷ•мқё

    мһ‘м„ұн•ң кёҖмқҙ backend м»Ён…Ңмқҙл„Ҳ лӮҙл¶Җмқҳ data.jsonм—җ мӢӨм ңлЎң м ҖмһҘлҗҳм—ҲлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ. м»Ён…Ңмқҙл„Ҳ лӮҙл¶ҖлЎң м§Ғм ‘ м ‘мҶҚн•ҙм„ң нҢҢмқј лӮҙмҡ©мқ„ м¶ңл Ҙн•ҙліҙм„ёмҡ”.

    л„ӨнҠёмӣҢнҒ¬ мҶҢмҶҚ нҷ•мқё

    frontendмҷҖ backendк°Җ лҸҷмқјн•ң л„ӨнҠёмӣҢнҒ¬м—җ мҶҚн•ҙ мһҲлҠ”м§Җ л„ӨнҠёмӣҢнҒ¬ мғҒм„ё м •ліҙлЎң нҷ•мқён•©лӢҲлӢӨ.

    мҷ„лЈҢ кё°мӨҖ¶

    н•ӯлӘ© нҷ•мқё
    мӮ¬мҡ©мһҗ м •мқҳ л„ӨнҠёмӣҢнҒ¬ мғқм„ұ мҷ„лЈҢ
    backend-service м»Ён…Ңмқҙл„Ҳ мӢӨн–ү (нҸ¬нҠё 5000)
    frontend м»Ён…Ңмқҙл„Ҳ мӢӨн–ү (8080:80)
    лёҢлқјмҡ°м Җм—җм„ң http://localhost:8080 м ‘мҶҚ м„ұкіө
    кёҖ мһ‘м„ұ нӣ„ нҷ”л©ҙ м¶ңл Ҙ нҷ•мқё
    backend лӮҙл¶Җ data.jsonм—җ лҚ°мқҙн„° м ҖмһҘ нҷ•мқё
    л‘җ м»Ён…Ңмқҙл„Ҳк°Җ лҸҷмқј л„ӨнҠёмӣҢнҒ¬ мҶҢмҶҚ нҷ•мқё
